File Paths

Examples

Full Path

 C:\PG2\Lectures\Week1\readme.txt
Includes the drive (C:), the directory paths (PG2\Lectures\Week1) and the filename (readme.txt).

Relative Path

 ..\Lectures\Week1\readme.txt
This path starts in the current working directory, moves up 1 folder (..\) then goes into the Lectures\Week1 folder to find the readme.txt file.
Relative Path movement:
".\" - means the current directory
"..\" - means to move up one directory on the path.

The current working directory is the directory that the application is currently using. Usually, this is the directory where the application is located.

Current Directory

readme.txt
When all you specify is the name of the file, your application will use the current working directory.

Extensions

The extension on your filename are the characters after the last '.' in the filename. NOTE: filenames can have any number of '.' in them. The extension is serves as information to the operating system about what could be in the file and which applications are associated with files of that type. Extensions are also information to people to help us know what could be in the file.

Working with Extensions

The Path class can be used to work with extensions (check them, get them, modify them).

Path.HasExtension

Will return true if the filepath has an extension and false if the filepath does not have an extension. 
string filePath = @"C:\PG2\Lectures\Week1\readme.txt";
bool hasExtension = Path.HasExtension(filePath);
Console.WriteLine($"{filePath} has extension? {hasExtension}");
This example prints C:\PG2\Lectures\Week1\readme.txt has extension? true

Path.GetExtension

Will return the extension including the '.' if it has an extension and String.Empty if it does not. 
string filePath = @"C:\PG2\Lectures\Week1\readme.txt";
string extension = Path.GetExtension(filePath);
Console.WriteLine($"{filePath} extension? {extension}");
This example prints C:\PG2\Lectures\Week1\readme.txt extension? .txt

Path.ChangeExtension

Will return a new string with the filePath where the extension has been changed. 
string filePath = @"C:\PG2\Lectures\Week1\readme.txt";
string newFilePath = Path.ChangeExtension(filePath, "gif");
Console.WriteLine($"new File Path: {newFilePath}");
This example prints new File Path: C:\PG2\Lectures\Week1\readme.gif
