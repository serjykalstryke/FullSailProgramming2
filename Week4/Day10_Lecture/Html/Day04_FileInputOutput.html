File I/O Overview

Pretty much every application uses File Input and Output to some degree. Learning how to work with files therefore is a vital developer skill.

To work with files, you need using System.IO; at the top of the file.

File I/O Classes

There are several classes in the System.IO namespace that we'll find useful:
  • File
    Helpful methods:
    • File.CreateText
    • File.ReadAllText
    • File.Exists
  • Directory
    Helpful methods:
    • Directory.CreateDirectory
    • Directory.EnumerateFiles, GetFiles
    • Directory.Exists
  • Path
    Helpful methods:
    • Path.ChangeExtension
    • Path.Combine

File Types

Below are some common data storage files.

CSV - Comman-separated values

"Thor,Captain America,Iron Man"

XML - eXtensible Markup Language

<?xml version="1.0"?>
<ArrogantSupers>
    <SuperHero Name="Thor" />
    <SuperHero Name="Captain America" />
    <SuperHero Name="Iron Man" />
</ArrogantSupers>

JSON - Javascript Object Notation

{
    "ArrogantSupers": [
        {"Name": "Thor"},
        {"Name": "Captain America"},
        {"Name": "Iron Man"}
    ]
}
