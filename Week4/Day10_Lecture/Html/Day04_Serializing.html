Serializing JSON

Serializing means to save the state of your objects. Your objects in code have data and methods. When serializing, you are only saving the data on those objects.

We are going to serialize some objects in JSON format. We'll use the Json.NET library to help us.

NOTE: There are different ways to serialize using Json.NET. I am showing you one way to do it.

Examples

To work with Json.NET, you need using Newtonsoft.Json; at the top of the file.

string jsonFilePath = Path.ChangeExtension(filePath, ".json");//ensure the file created has a .json extension
using (StreamWriter sw = new StreamWriter(jsonFilePath))
{
    using (JsonTextWriter jtw = new JsonTextWriter(sw)) //use a nested using statement for the JsonTextWriter
    {
        JsonSerializer serializer = new JsonSerializer();
        serializer.Formatting = Formatting.Indented; //this makes the file more readable for humans. Don't do this if you're passing the data across the internet.
        serializer.Serialize(jtw, bats);
    }
}
You can replace bats on the .Serialize method call with whatever object you want to save. Json.NET will handle how to serialize the object's data.
