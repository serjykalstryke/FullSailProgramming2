Parameters by Reference

Pass by Reference == ALIAS

When you choose to pass a variable by reference to a method, the method does not create a separate variable. Instead, the parameter in the method is just another name for the variable that is used when calling the method.

Examples

Here's a method that uses 2 reference parameters. The method will assign a value to x and y. However, those parameters are just aliases for the variables used when calling the method. 
static void GetPosition(ref int x, ref int y)
{
    x = 12;
    y = 25;
}
When we call the GetPosition method, we pass 2 variables by reference to the method. Those variables will now have the values that GetPosition sets. NOTE: when calling the method, you must use the ref keyword. 
int xpos = 0, ypos = 0;
GetPosition(ref xpos, ref ypos);
After calling the method, xpos will be 12 and ypos will be 25.
