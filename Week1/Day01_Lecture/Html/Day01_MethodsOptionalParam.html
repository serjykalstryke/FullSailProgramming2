Optional Parameters

Optional Parameters

Optional parameters are parameters that are given a default value if they are not passed in by the code that calls the method.
If a value is passed in for the optional parameter, that value will be used instead of the default value.
All optional parameters must appear at the end of the parameter list.

To make a parameter optional, assign a value to the parameter in the parameter list. (see the first example below)

Examples

This method has 1 required parameter and 1 optional parameter. 
public static string PostFix(string fileName, int postFixNumber = 1)
{
    return fileName + "-" + postFixNumber;
}
When calling the method, the second parameter is optional. If a value is not passed in, 1 will be assigned to the parameter. 
string newFileName = PostFix("outputFile"); //uses the optional value of 1
string otherFile = PostFix("outputFile", 5); //passes 5 in to the parameter
newFileName will have a value of outputFile-1.
otherFile will have a value of outputFile-5.
