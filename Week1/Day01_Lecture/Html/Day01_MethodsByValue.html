Parameters by Value

Pass by Value == COPY

Examples

Here's a method that adds 2 numbers. 
static int Add()
{
    return 5 + 2;
}
However, this is not a very useful method. It always returns 7. It would be great if you could create a method that would add any 2 numbers that I want it to. Therefore, I need to define 2 variables as parameters that can be changed. 
static int Add(int number1, int number2)
{
    return number1 + number2;
}
This method now will add any 2 integers that are passed to it when the method is called. 
int sum1 = Add(5, 2);
int sum2 = Add(13,5);
The first line will copy 5 into the number1 variable and copy 2 into the number2 variable. The second line will copy 13 into the number1 variable and copy 5 into the number2 variable.
