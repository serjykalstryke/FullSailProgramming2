Cloning arrays and Lists

Creating Lists from Arrays

If you have an array but need a List instead (ex: a method requires a List parameter), you can create a List from the array using one of the following ways:
  • call ToList on the array
  • Pass the array to the List constructor
  • Add each item from the array to a new list

Examples

call ToList on the array

NOTE: To use the ToList method, you need using System.Linq; at the top of the file.

int[] nums1 = new int[] {5, 4, 4, 2, 1};
List<int> nums2 = nums1.ToList();

Pass the array to the List constructor

int[] nums1 = new int[] {5, 4, 4, 2, 1};
List<int> nums2 = new List<int>(nums1);

Add each item from the array to a new list

int[] nums1 = new int[] {5, 4, 4, 2, 1};
List<int> nums2 = new List<int>();
for(int i=0; i < nums1.Length; i++)
    nums2.Add(nums1[i]);

Cloning a List

If you simply assign one list to a new list variable (List<int> nums2 = nums1;), you have 2 variables pointing to the same list.
If you change something about the nums2 list, you are actually changing the list that nums1 points to.


Sometimes what you want to do is create a copy or a clone of a list.
Changing the clone will not affect the original list.

You can create a copy of a List using one of the following ways:

  • call ToList on the List
  • Pass the List to the List constructor
  • Add each item from the List to a new list

call ToList on the array

NOTE: To use the ToList method, you need using System.Linq; at the top of the file.

List<int> nums1 = new List<int>() {5, 4, 4, 2, 1};
List<int> nums2 = nums1.ToList();

Pass the array to the List constructor

List<int> nums1 = new List<int>() {5, 4, 4, 2, 1};
List<int> nums2 = new List<int>(nums1);

Add each item from the array to a new list

List<int> nums1 = new List<int>() {5, 4, 4, 2, 1};
List<int> nums2 = new List<int>();
for(int i=0; i < nums1.Length; i++)
    nums2.Add(nums1[i]);

Try it!

  1. On line 12, clone the JLA list.
  2. In the cloned list, remove Aquaman.
  3. Then after the foreach, print the cloned list with a title of New and Improved Justice League.

public static void Main(string[] args)
{
    List<string> JLA = new List<string>() {"Batman", "Flash", "Superman", "Wonder Woman", "Aquaman" };
    List<string> clone = JLA.ToList();
    clone.Remove("Aquaman");
    Console.WriteLine("Original Justice League");
    foreach(string super in JLA)
        Console.WriteLine(super);

    Console.WriteLine("New and Improved Justice League");
    foreach(string super in clone)
        Console.WriteLine(super);
}
