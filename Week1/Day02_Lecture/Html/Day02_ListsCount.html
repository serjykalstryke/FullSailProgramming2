Lists - Count and Capacity

The array has the Length property to tell you how big the array is. Since arrays have a static size, that length does not change.
However, that length does not tell you how many items have been added to the array.

The List<T> has 2 properties that convey information about the items in the list: Count and Capacity.

Capacity

Internally, the List<T> class stores the data in an array.
The Capacity tells you how big that array is. It is basically the Length of that internal array.

Count

The Count property tells you how many items have been added to the List.
The Count will always be <= the Capacity.
When accessing the items in a List, you can only access the indexes from 0 to (Count - 1).
Therefore, the majority of the time, you will use Count when accessing items in a List or looping over a List.
