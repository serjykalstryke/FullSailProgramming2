List<T>

The List type in C# is a collection that auto-resizes itself. Sometimes these are referred to as dynamic arrays.
Internally, the List class uses an array to store the data. So it has the same pros/cons of the array.
The benefit to the developer is that the code needed to resize the array has been written already and is handled automatically.

Examples

NOTE: To use the List class, you need using System.Collections.Generic; at the top of the file.

Creating a List

The List<T> class is a generic class.
To use it for a variable, you must replace the T with whatever type you want to store in the List.
This example is creating 2 different Lists. One holds strings and the other holds ints. 
List<string> names = new List<string>() { "Bruce", "Batman", "The Dark Knight"};
List<int> highScores = new List<int>();

Adding items to a List

You can add items in the initializer when you create a List or you can use the Add method. 
//add items in the initializer when you create the list object
List<string> names = new List<string>() { "Bruce", "Batman", "The Dark Knight"};

//add items with the add method
names.Add("Clark");
names.Add("Superman");
