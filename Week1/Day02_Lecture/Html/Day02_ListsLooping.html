Lists - Looping

You can use a for loop or a foreach loop to access each of the items in a List.

Examples

for loop

Use the Count property when looping with a for loop.
Use the [ ] indexer to access the values stored in the List. 
List<int> nums = new List<int>() {5, 4, 3, 2, 1};
for(int i = 0; i < nums.Count; i++)
{
    //modify the value stored in the List
    nums[i] *= 2;

    //access the value stored in the List
    Console.WriteLine(nums[i]);
}

foreach loop

The foreach loop will give you each value in the List and assign it to the foreach variable.
In this example, the foreach variable is called number. Each time through the loop, the next value in the List will be assigned to number.

NOTE: Modifying the foreach variable will NOT modify the value stored in the List.

List<int> nums = new List<int>() {5, 4, 3, 2, 1};
foreach(int number in nums)
{
    number *= 2; //this will NOT modify the value in the List
    Console.WriteLine(number);
}
