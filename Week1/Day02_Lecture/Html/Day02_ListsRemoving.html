Lists - Removing

There are two main ways to remove items from a List: Remove and RemoveAt.

NOTE: When removing a value using either approach, all the items to the right of the item in the list are shifted 1 spot to the left.
EX: if you remove 3 from List<int> nums = new List<int>() {5, 4, 3, 2, 1};, the values 2 and 1 will be shifted one spot to the left in the list.
The resulting list will be {5, 4, 2, 1}.

Examples

Remove

The Remove method takes 1 parameter, the item that you want to remove.
List will start at the beginning of the List and search until it finds the item.
List will only remove the first item that matches. Once it finds the item and removes it, the method returns. 
List<int> nums = new List<int>() {5, 4, 4, 2, 1};
nums.Remove(4);
After Remove is called, the nums List will have these values: 5, 4, 2, 1.

RemoveAt

The RemoveAt method takes 1 parameter, the index of the item to remove. 
List<int> nums = new List<int>() {5, 4, 4, 2, 1};
nums.RemoveAt(4);//to remove the 1 value, pass in the index of 4.
After RemoveAt is called, the nums List will have these values: 5, 4, 4, 2.
