Arrays


Arrays are blocks of data stored contiguously in memory. So if you create an array to hold 4 integers, each integer will be right next to each other in memory.
To lookup a value in the array, you must use an index into the array. Arrays are zero-based indexed collections. That means the first index is 0.
So, for the int array that holds 4 integers, the index range is 0 to 3.

Examples

How to create an array 
int[] highScores = new int[10]; //an array that holds 10 ints.
string[] JLA = new string[] {"Batman", "Superman", "Wonder Woman"}; //a string array that holds 3 strings
How to loop over an array NOTE:
  • use the Length property to know how many items are in the array.
  • use the indexer [ ] to access an element in the array.
for(int i=0;i<JLA.Length;i++)
{
    Console.WriteLine(JLA[i]);
}
How to assign a value to an item 
JLA[1] = "Aquaman";
