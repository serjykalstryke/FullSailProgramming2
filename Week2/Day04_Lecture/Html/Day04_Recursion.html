Recursion Overview

Recursion happens when a method calls itself. This will create a loop

Most loops have a condition that controls the loop. For instance, for(int i=0;i < 100;i++) the part of the for loop (i < 100) controls the loop. If the condition is false, the loop stops. The loop will continue as long as the condition is true.

Recursion is another kind of loop. Therefore, you want to include code in the method that will control the recursive loop. The code that controls a recursive loop is called an exit condition.

Examples

This for loop will loop 100 times printing the numbers 0-99. 
for(int i=0;i < 100;i++)
{
    Console.WriteLine(i);
}
An equivalent recursive loop... 
int i = 0;
RecursiveMethod(i);

static void RecursiveMethod(int i)
{
    if(i < 100) //the exit condition
    {
        Console.WriteLine(i);
        RecursiveMethod(i+1);//the recursive call
    }
}

Try it!

Write code in RandomRecursive to print the random number and call itself while the random number is not 50. 

public static void Main(string[] args)
{
    RandomRecursive();
}
static Random rando = new Random();
static void RandomRecursive()
{
  int num = rando.Next(100);
  Console.WriteLine(num);
  if(num != 50)
    RandomRecursive();
}
