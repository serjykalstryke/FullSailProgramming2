Comparing Strings

Strings can be easily compared for equality using == in conditions. However, less than (<) and greater than (>) won't work with strings in C#.

To compare strings in C#, we use the CompareTo method on the string.

CompareTo returns an int. Here are the meanings of the numbers that it can return:

  • -1: less than
  • 0: equal to
  • 1: greater than

Examples

string s1 = "Batman", s2 = "Aquaman";
int compResult = s1.CompareTo(s2);
if(compResult < 0) 
    Console.WriteLine($"{s1} is LESS THAN {s2}");
else if(compResult == 0) 
    Console.WriteLine($"{s1} is EQUAL TO {s2}");
else if(compResult > 0) 
    Console.WriteLine($"{s1} is GREATER THAN {s2}");
