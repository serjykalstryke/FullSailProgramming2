Dictionaries - Removing

To remove a key and its associated value from a Dictionary, you call the Remove method and pass in the key that you wish to remove. This method returns true or false depending on if the key is found and removed. NOTE: the value for the key will also be removed from the Dictionary.

You do NOT need to check if the key is available before calling Remove. If the key is not found, false will be returned.

Examples

How to remove a key and its value

string itemToRemove = "Chicken Nuggets";
bool wasRemoved = menu.Remove(itemToRemove);
if(wasRemoved)
    Console.WriteLine($"{itemToRemove} was removed from the menu.");
else
    Console.WriteLine($"{itemToRemove} was not found.");
