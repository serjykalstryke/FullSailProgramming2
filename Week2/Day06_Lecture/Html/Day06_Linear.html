Linear Search

The linear search algorithm allows the user to search for a specific value.
The algorithm:
  1. Start at the beginning of the list
  2. Stop when the item is found OR the end of the list is reached
  3. Return the index of the found item OR -1 if the item is not found.
If the item is not found, -1 is returned because -1 is an invalid index and indicates to the calling code that the item was not found.

Performance

The worst-case performance for linear search is O(N) - linear.
