Dictionaries - Creating and Adding

NOTE: To use the List class, you need using System.Collections.Generic; at the top of the file.

Dictionary<TKey,TValue>

Dictionaries are generic classes. To create a Dictionary variable, you need to set the type of the key and the type of the value. Because they are defined separately, the type of the key does not have to be the same as the type of the value.

Examples

Create a Dictionary:

To create a Dictionary to associate Student IDs to Student records:
Dictionary<string,StudentRecord> studentInfo = new Dictionary<string,StudentRecord>();

Add items in the initializer:

Dictionary<string,float> menu = new Dictionary<string,float>()  //don't put a ; here
{
    //{ key, value }
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  //put the ; here

Add a key and value using the Add method:

menu.Add("French Fries", 0.99F);
menu.Add("Soda", 0.99F);

The Add method will throw an exception if the key is already in the dictionary.

Add a key and value using [ ]:

menu["Chicken Nuggets"] = 3.99F;

[ ] will NOT throw an exception if the key is already in the dictionary. It will overwrite the value for the key.
