Dictionaries

A Dictionary is an associative collection. That means that the key (the item used to lookup a value) is associated with the value. For arrays and Lists, the index (the item used to lookup a value) has nothing to do with the value in the array/List.

For Dictionaries, the key is usually part of the value. For instance, your Student ID is part of your Student record at Full Sail. Your Student ID is also used to lookup your Student record.

Performance

Lookup Performance

The performance of using an index to lookup a value in an array/List: O(1) constant.

Dicionaries don't use indexes. Rather, they use keys to lookup values. The performance of using key to lookup a value in Dictionary: O(1) constant.

Searching Performance

The performance searching an array/List for a value: O(N) linear. (Linear is worse than constant)

Dicionaries use keys to find values. The performance of using key to find a value in Dictionary: O(1) constant.
