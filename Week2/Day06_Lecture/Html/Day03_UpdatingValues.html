Dictionaries - Getting and Updating Values

There are 2 ways to get the value for a key:
  • TryGetValue(key, out value)
  • [key]

Only use [key] to get a value if you know that the key is in the dictionary. If the key is not in the dictionary, [key] to get a value will throw an exception!

To update a value for a key, use the [ ] on the left-hand side of an assignment operator.

Examples

TryGetValue

Dictionary<string,float> menu = new Dictionary<string,float>()  
{
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  
string keyToFind = "Cheesy Hamburger";
bool isFound = menu.TryGetValue(keyToFind, out float price);
if(isFound)
    Console.WriteLine($"{keyToFind} costs {price:C2}");
This example will print Cheesy Hamburger costs $2.99 because it was in the dictionary.

[key]

Dictionary<string,float> menu = new Dictionary<string,float>()  
{
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  
string keyToFind = "Cheesy Hamburger";
float price = menu[keyToFind]; //NOTE: if you're not sure if the key is in the dictionary, use ContainsKey to check first otherwise it'll throw an exception
Console.WriteLine($"{keyToFind} costs {price:C2}");
This example will print Cheesy Hamburger costs $2.99.

Updating a Value

Dictionary<string,float> menu = new Dictionary<string,float>()  
{
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  
string keyToFind = "Cheesy Hamburger";
float price = menu[keyToFind]; //NOTE: if you're not sure if the key is in the dictionary, use ContainsKey to check first otherwise it'll throw an exception
menu[keyToFind] = price + 1F;
Console.WriteLine($"{keyToFind} now costs {menu[keyToFind]:C2}");
This example will print Cheesy Hamburger now costs $3.99.
