OOP - Extension Methods

Extension Methods are a way in C# where you can add methods to a class without modifying the class and without deriving frmo the class.

How to add an extension method

  1. Create a static class
  2. Add a static method where the first parameter sets the 'this' parameter for the method. The 'this' parameter determines what class the method is attached to.

You cannot add an extension method to a static class because you need an instance for the 'this' parameter.

Examples

Add a Print extension method to the List<T> type. 
public static class Extensions
{
    public static void Print<T>(this List<T> items)
    {
        foreach(var item in items)
            Console.WriteLine(item);
    }
}
The Print method is now added to the List class.
items.Print(); //if items is a List<T>
