OOP - Overloading

Overloading allows you to change the behavior of a method based on how the method is called.

Ways to Overload

  • The number of parameters is different.
    • public int Add(int n1, int n2) { return n1 + n2; }                
public int Add(int n1, int n2, int n3) { return n1 + n2 + n3; }
  • The types of parameters are different.
    • public int Add(int n1, int n2) { return n1 + n2; }                
public double Add(double n1, double n2) { return n1 + n2; }
  • The order of parameters are different.
    • public string Concat(string prefix, int num) { return prefix + n; }             
public string Concat(int num, string prefix) { return n + prefix; }

You cannot overload a method on just the return type. The parameters must be different.
