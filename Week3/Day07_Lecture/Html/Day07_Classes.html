OOP - Classes

Classes usually have their own file. Do not put multiple classes in the same file.

Creating a Class

To create a class, use the class keyword followed by the name of the class. 
class Car
{
}

Class Accessibility

You can add either public or internal to the class to control its accessibility.

internal is the default accessibility given to classes. Note: the Car class example above has internal applied to it.

If you want the class to be used outside of the project (assembly), you must mark it as public.

public class Car
{
}
