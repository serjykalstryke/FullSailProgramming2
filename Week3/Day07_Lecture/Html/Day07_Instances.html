OOP - Instances

To use a non-static class, you must first create an instance of the class. To do so, you must call a constructor using the new keyword. 
public class Car
{
    public string Model {get;set;}
    public string Make {get;set;}
    public Car(string make, string model) { Make = make; Model = model; }
}
The Car class has 1 constructor that requires 2 string parameters. To create an instance of Car, you would need to pass in 2 strings to the constructor: 
Car myCar = new Car("Ford", "Mustang");
