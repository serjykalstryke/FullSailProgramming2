OOP - Constructors

Constructors are special methods that are meant to initialize the instances of your class.

The rules of constructors:

  • Constructors cannot have a return type (not even void).
  • Constructors must have the same name as the class.
  • If you do not provide a constructor for your class, the compiler will give you a default constructor.

Examples

public class Car
{
    public int Year { get; private set; }

    //constructors
    public Car()
    {
        Year = 0; //assign a default value to the Year property
    }
    public Car(int year)
    {
        Year = year; //assign the year parameter to the Year property
    }
}
In this example, the Car class has 2 constructors.
  • The first constructor is a default constructor. It takes no parameters.
  • The second constructor is an overloaded constructor that takes 1 int parameter. Use the year parameter to assign a value to the Year property.
How to use the first constructor: 
Car myCar = new Car();
How to use the second constructor that takes a year parameter: 
Car myCar = new Car(1988);//the year parameter will have the value of 1988
