OOP - Methods

Methods are just blocks of code. They are the behavior of the class (what the class does).

Static vs Instance Methods

If the static keyword is applied to a method, then the method is attached at the class level. The only way to access the static method is through the class name.
Instance method are those method without the static keyword. They are only visible on the instances of the class and can only be accessed through the instance.

Instance Methods

Instance methods are used on an instance of the class. They have a hidden parameter called this that points to the instance the method was called on. 
public class Car
{
    public string Model {get;set;}
    public string Make {get;set;}
    public Car(string make, string model) { Make = make; Model = model; }
    public void Honk() { Console.WriteLine($"My {this.Model} goes HONK HONK!!"); }
}
The Honk method is an instance method. You call it on an instance of the Car class. 
Car myCar = new Car("Tesla", "Cybertruck");
Car yourCar = new Car("Ford", "Mustang");
myCar.Honk(); //prints 'My Cybertruck goes HONK HONK!!'
yourCar.Honk(); //prints 'My Mustang goes HONK HONK!!'
The Honk method is called on the myCar instance. It prints the Model of myCar because 'this' is passed in which points to myCar.

Static Methods

Static methods are attached at the class level, not the instance level. Therefore, there is NO this parameter passed to static methods because you call static methods with the name of the class, not the instance. 
public class Car
{
    private static int _numberOfCarsBuilt = 0;
    public string Model {get;set;}
    public string Make {get;set;}
    public Car(string make, string model) { _numberOfCarsBuilt++; Make = make; Model = model; }

    public static void CarsBuilt() 
    { 
        Console.WriteLine($"The number of cars built: {_numberOfCarsBuilt}"); 
    }
}
Because the CarsBuilt method is static, we call the method using the name of the class, Car. 
Car.CarsBuilt();

Static methods cannot access instance members. In the Car class, the CarsBuilt method cannot access Make and Model because they are instance properties. Static methods can only access static members.
