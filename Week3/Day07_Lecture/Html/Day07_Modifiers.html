OOP - Access Modifiers

For the members inside of a class (fields, properties, methods), you can assign an access modifier that controls the accessibility of the item (who can see it).
The commonly used access modifiers:
  • public
  • private
  • protected

Public

Public means that all code can see it -- visible to everyone. 
public class Car
{                
    public ConsoleColor color = ConsoleColor.Green; 
}
Everyone can see and edit the color field.

Private

Private means that ONLY> this class can see it -- visible to no one else. 
public class Car
{                
    private ConsoleColor color = ConsoleColor.Green; 
}
Only the code inside the { } of the Car class can see and edit color.

Protected

Protected means that this class and all other classes that derive from this class can see it. 
public class Car
{                
    protected ConsoleColor color = ConsoleColor.Green; 
}
public class ElectricCar : Car
{
    public void CarInfo() {
        Console.WriteLine($"My color is {color}.");
    }
}
Only the code inside the { } of the Car class and derived classes (ElectricCar) can see and edit color.
