How to work on the lab

  1. Find the location on your hard drive

    Find the location on your hard drive where you cloned the GitHub repo. By default, Visual Studio will clone it in this location: C:\Users\<your windows username>\source\repos.
    Look in the Week3\Lab3 folder.

  2. Open the Lab3.sln

    Double-click the sln file to open the solution in Visual Studio.
