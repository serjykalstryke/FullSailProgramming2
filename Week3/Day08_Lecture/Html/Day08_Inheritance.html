OOP - Inheritance

Inheritance is a way to reuse code from an existing class and to add new functionality.

Deriving

When you create a new class and want to derive (inherit) the functionality of another class, you add the base class name after the name of your derived class. 
public class A {  }
public class B : A  {    }
Class B derives from A (B : A).
Everything in class A is now part of class B.

In C#, your class can only have 1 base class. You cannot derive from multiple classes.
However, you can implement multiple interfaces.
