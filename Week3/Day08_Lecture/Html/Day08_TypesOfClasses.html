OOP - Types of Classes

Other Types of Classes

Besides just a standard class, here are some other kinds of classes:

  • Abstract Classes

    A class marked with the keyword abstract. You cannot create instances of abstract classes. The purpose of abstract classes is to serve as a base class for other classes. public abstract class A { }

  • Static Classes

    A class marked with the keyword static. You cannot create instances of static classes. Everything defined in the static class must be marked as static. See below for more info on static classes. public static class B { }

  • Nested Classes

    Defining a class inside of another class. The advantage is the nested class has access to the private members of the containing class.

  • Sealed Classes

    A class marked with the keyword sealed. This makes it so that you cannot derive a class from the sealed class. public sealed class D { }

  • Partial Classes

    A class marked with the keyword partial. This makes it so that you split the definition of a class across 2 or more files. public partial class E { }
