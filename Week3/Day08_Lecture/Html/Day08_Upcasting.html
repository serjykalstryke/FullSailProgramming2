OOP - Upcasting

Upcasting is when you take a variable of a derived class type and you cast it to a variable of a base class type.
BaseClassType variableB = variableA; //if variableA is a derived class of BaseClassType

Upcasting is always safe. The compiler knows that the derived class inherits from the base class so it can easily cast to the base type, even if the base type is multiple levels up the hierarchy.

Examples

Upcasting

public class Person  {  }
public class Superhero : Person  {  }

Superhero bats = new Superhero();
Person bruce = bats; //this is upcasting and SAFE because Superhero derives from Person

When you upcast, you are NOT creating a new object. You are just looking at the SAME object through a different "lens" -- the base class type.
