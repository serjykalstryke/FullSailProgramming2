Console: Memory

In .NET, there are 2 kinds of data types:

  • value types

    The data for value types are stored in the stack.
    These include the build-it data types like bool, int, float, double, etc. This does NOT include string.
    User-defined structures are also included. The memory allocated for the value type is dependent on the data type.

  • reference types

    These are the classes and arrays.
    The contents (data) for these are stored in the heap.
    The memory allocated for reference type variables are dependent on the application. If the application is 32-bit, 4 bytes will be allocated. If the application is 64-bit, 8 bytes will be allocated.

stack vs heap

  • Stack memory is allocated next to each other in the order that the variables are declared.
  • Heap memory is allocated but the data is NOT guaranteed to be next to each other.

Examples

int iVar = 5;
float fVar = 5.0F;
int[] ids = new int[4] {4,3,1,9};
