Console: Output

Cursor

When printing items to the console, the cursor location determines where they are printed. There are several methods that you can use to position the cursor.
  • Console.CursorLeft Allows you to get or set the X position (horizontal) of the cursor.
  • Console.CursorTop Allows you to get or set the Y position (vertical) of the cursor.
  • Console.SetCursorPosition(x,y) Allows you to set the X and Y positions of the cursor. 
int xPos = Console.CursorLeft; //get the horizontal position
Console.CursorLeft = xPos + 5; //set the horizontal position. Moves the cursor right by 5 columns.

int yPos = Console.CursorTop; //get the vertical position
Console.CursorTop = yPos + 5; //set the vertical position. Moves the cursor down by 5 rows.

Console.SetCursorPosition(3,12); //moves the cursor to the 3rd column and 12th row.

Window Size

The Console window has a specific size. You can retrieve that size using the WindowWidth and WindowHeight properties of the Console class. 
int width = Console.WindowWidth; 
int height = Console.WindowHeight;

The range of the cursor within the Console window is (0 to Console.WindowWith-1) wide by (0 to Console.WindowHeight-1) high.

If you try to set the x position of the cursor >= WindowWidth, it will cause an exception. EXAMPLE: 

Console.CursorLeft = Console.WindowWidth; //this will throw an exception

Colors

You can control the foreground color and the background color of the items printed to the Console. NOTE: setting the colors change what color is used when something is printed after setting the color. It will not affect the color of items already printed.

Foreground Color

The foreground color determines the color of any text printed. 
Console.ForegroundColor = ConsoleColor.Yellow; 
Console.WriteLine("Batman!");
This example will print "Batman!" in yellow.

Background Color

The Background color determines the color of the behind the text that is printed. 
Console.BackgroundColor = ConsoleColor.DarkBlue; 
Console.WriteLine("Batman!");
This example will print "Batman!" on a dark blue background.

Once you are done printing something in the colors you choose, it is best to reset the colors to the defaults.
Console.ResetColor();
