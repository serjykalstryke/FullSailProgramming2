GitHub: Cloning the Repo

Cloning the Repo

Cloning the repo means making a local copy of your GitHub repo on your laptop/desktop.

You will need the link to your GitHub repo. You can find the link on the Classroom Invitation success page. NOTE: if you click the Classroom Invitation link again, it will show the success page again with the link.

1. Open Visual Studio

When Visual Studio opens, you'll see a dialog like this:

Click the "Clone a repository" button.

2. Select a location

On the dialog that shows next...
  1. Paste your GitHub repo link in the "Repository location" textbox.
  2. In the Path textbox, select (or find using the ... button) a location on your harddrive. This will be where the repo is placed on your machine.

Once this completes, the repo will have been cloned to your local machine. Remember the local path you entered in step 2 so that you can easily find your code again.

If this step fails, check the Troubleshooting page for possible solutions.
