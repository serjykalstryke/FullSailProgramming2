GitHub: Classroom Assignment

1. Click the Invitation Link

You'll receive a GitHub Classroom invitation link in the GitHub assignment in FSO.

When you click the invitation link, you'll see something like this:


Click the green Accept this assignment button.

2. Copy the repo link

Once GitHub finishes creating your repo, you'll see a screen like this:

Copy the URL in the blue box on the success page. That is the link to your repo. Put that in the feedback of the GitHub assignment in FSO.
