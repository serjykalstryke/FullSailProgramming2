GitHub: Troubleshooting

You might encounter some problems working with GitHub. Below are some of the more common problems.

If you still encounter problems, reach out to your course director as soon as possible.

Problem: my "Commit all" button is grayed out!

Solution #1: Restart Visual Studio

Sometimes Visual Studio's Git Changes tab does not refresh. Closing and reopening Visual Studio may fix the problem.

Solution #2: Enter a commit message

The commit message is required. Be sure to enter a message in the text box that is above the Commit All button.

Solution #3: Configure GitHub


If you see a yellow box at the top of the Git Changes tab:
  1. Click the Configure link OR Open the Git settings dialog
  2. Ensure that your username and email address are the same as you used in GitHub

Problem: my GitHub repo cannot be found!

Solution: Fix GitHub Credentials

If you get a message saying that your GitHub repo cannot be found, it is likely that your machine is using different GitHub credentials.
  1. Open the account settings dialog. In the search bar at the top of Visual Studio, search for "account" and select the Account Settings.
  2. Remove and re-add your GitHub credentials. In the Account Settings dialog box,
    • Click remove next to the GitHub account that is listed
    • Click the +Add button to re-enter your GitHub credentials.
